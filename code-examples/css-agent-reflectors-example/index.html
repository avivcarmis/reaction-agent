CSS Agent Custom Reflectors Examples

The behavior and HTML attribute parsing of the CSS Agent can be customized by setting custom reflectors when calling the CSSAgent.start function.

In this example we will configure 2 reflectors only, big which reflects viewport with width above or equal to 768 pixels (typically, everything but mobile device), and small which reflects viewports with width of up to 768 pixels (typically mobile devices). Lets check it out.

The Source Code

<script type="text/javascript"> CSSAgent.start([ {name: 'small', min: null, max: 768}, {name: 'big', min: 768, max: null} ]); </script> <style type="text/css"> .red-box { background: red; height: 200px; margin: 20px 0; } </style> <div class="red-box" data-ca-big="background: yellow;"> This box is yellow on <i>big</i> viewport, red otherwise </div> <div class="red-box" data-ca-small="background: yellow;"> This box is yellow on <i>small</i> viewport, red otherwise </div> <div class="red-box" data-ca-small-hover="background: yellow;"> This box is yellow <b>on hover</b> on <i>small</i> viewport, red otherwise </div>

The Result

This box is yellow on big viewport, red otherwise
This box is yellow on small viewport, red otherwise
This box is yellow on hover on small viewport, red otherwise

Conclusions & Notes

  • You may resize the browser window in order to view the different media query reactions.
