DOM Agent Events Examples

In addition to the DOM elements constructor concept, the DOM Agent provide us with another powerful mechanism of triggering events of element inner DOM change.

This could be useful when we would like to receive updates and perform some actions when some element inner HTML gets changed.
The example below demonstrates a situation when we would like to show an alert message whenever the #example element gets changed. This example situation can be extended into a real situation where we have a custom built scrollbar, and we want to update it whenever inner changes occur for example.

The Source Code

<style> #example > * { min-height: 40px; margin: 10px 0; background: yellow; } #example > *:nth-child(1) { background: red; } #example > *:nth-child(2) { background: blue; } #example > *:nth-child(3) { background: green; } #example > *:nth-child(4) { background: black; } #example > * > * { background: grey; margin: 10px; min-height: 20px; } #example > * > * > * { background: white; margin: 10px; min-height: 10px; } </style> <script type="text/javascript"> DOMAgent.init(); $(function() { $("#example").on('domagentchange', function() { alert('changed!'); }); }); function appendAChild() { $("#example").append('<div>'); } function prependChild() { $("#example").prepend('<div>'); } function empty() { $("#example").empty(); } function prependToChild() { $("#example > *").eq(0).append('<div>grandchild</div>'); } function prependToGradchild() { $("#example > * > *").eq(0).append('<div>grand grandchild</div>'); } </script> <div id="example"> <div></div> </div> <div> <button class="btn btn-default" onclick="appendAChild()">Append Child</button> <button class="btn btn-default" onclick="prependChild()">Prepend Child</button> <button class="btn btn-default" onclick="empty()">Empty</button> <button class="btn btn-default" onclick="prependToChild()">Prepend To Child (Only if child exists)</button> <button class="btn btn-default" onclick="prependToGradchild()">Prepend To Grandchild (Only if grandchild exists)</button> </div>

The Result

Conclusions & Notes

  • Do not get confused from the different colors, the main idea of the example is to demonstrate that the event listener react to any inner changes no matter at what depth they reside.
  • Note that in order to eliminate unnecessary use of resources, the DOM events module will not perform unless we explicitly calls the DOMAgent.init() function.
