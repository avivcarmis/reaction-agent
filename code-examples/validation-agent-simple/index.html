Validation Agent Simple Usage

The Validation Agent is mainly used for client side validation of form fields. Note that client side validations are only matter of good UX practices, but not for security measures - for that end, use must perform server side validations.

The Validation Agent is triggered and set throw jQuery as shown in the example.

Below an example of simply applying validation agent on two form fields - username and password.

The Source Code

<style> .error { min-height: 30px; background: #EADBDB; line-height: 30px; margin: 10px; } </style> <script type="text/javascript"> $(function() { $("#username").validationAgent(/^[a-zA-Z0-9]+$/, function() { $("#username-error").empty(); }, function() { $("#username-error").html('Username may contain only alpha numeric characters'); }); }); $(function() { $("#password").validationAgent(/^(?=.*\d)(?=.*[a-z])(?=.*[A-Z])[0-9a-zA-Z]{8,}$/, function() { $("#password-error").empty(); }, function() { $("#password-error").html('Password may contain at least one digit, one lower case letter, one upper case letter and at least 8 total characters'); }, {validateType: false}); }); </script> <form> <input class="form-control" id="username" type="text" name="username" value="" placeholder="Your username here"> <div id="username-error" class="error"></div> <input class="form-control" id="password" type="password" name=" password" value="" placeholder="Your password here"> <div id="password-error" class="error"></div> </form>

The Result

Conclusions & Notes

  • Try to interact with the form, the username field may contain only alphanumeric chars (a-z, A-Z, 0-9). Try to type in space character for example and see what happens.
  • Note that the password field, on the other hand, will not block you from typing anything, but only validate you on blur event, meaning when you focus on another element in the page, or click somewhere outside the password field box. This is accomplished by setting the Validation Agent settings with validateBlur and validateType to true or false.
