Validation Agent And DOM Agent Integration

The best way to use the validation agent is through DOM element custom data attributes. In order to write a DOM element that gets validated automatically, we can embed some DOM Agent constructor code to assign the Validation Agent for all elements containing certain data.

The Source Code

<script type="text/javascript"> DOMAgent.add('[data-validation]', function() { var regexp = new RegExp($(this).attr('data-validation')); $(this).validationAgent(regexp, function() { $(this).popover('destroy'); }, function() { $(this).popover('show'); }); }); </script> <form> <input class="form-control" type="text" name="username" value="" placeholder="Your username here" data-validation="^[a-zA-Z0-9]+$" data-content="Username may contain only alpha numeric characters"> <input class="form-control" type="password" name=" password" value="" placeholder="Your password here" data-validation="^[a-zA-Z0-9]+$" data-content="Password may contain only alpha numeric characters"> </form>

The Result

Conclusions & Notes

  • Try to interact with the form fields, they both allow only alphanumeric chars (a-z, A-Z, 0-9).
  • Note that once the DOM Agent initialize code snippet is embedded in your js, you do not need to use anymore javascript for that end, all elements containing data-validation attribute gets validated.
