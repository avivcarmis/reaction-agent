Scroll Agent Breakpoints Example

The Scroll Agent breakpoints are a simple mechanism. It is built on the idea of triggering handlers whenever a certain point is reached by the window scroll top value.

The example below demonstrates a few different ways of registering breakpoints, varying from explicit value, dynamic function, and element reached.

The Source Code

<script type="text/javascript"> var sa = new ScrollAgent(); sa.add(0, function() { $("#monitor").html(0); }); sa.add(100, function() { $("#monitor").html(100); }); sa.add(function() { return $(window).height(); }, function() { $("#monitor").html('window'); }); sa.add("#example", function() { $("#monitor").html('element'); }); </script> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <div id="example" style="height: 40px; background: red;">Element</div> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <br><br><br><br> <div id="monitor" style="position: fixed; bottom: 20px; right: 20px; height: 70px; line-height: 70px; width: 70px; text-align: center; background: rgba(0,0,0,.4); color: white;"></div>
