Development Utilities

The Reaction Agent comes with a build in set of utility methods to ease the client side development of web applications.

Is Defined
isDefined(variable);
What?

Receive a variable and returns whether or not it is defined. (Much like PHP's isset function).

Returns?

True if the variable is defined, false otherwise.

Example?
var x; var y = 2; isDefined(x); // false isDefined(y); // true isDefined(someNotDeclaredVariable); // false
Not Defined
notDefined(variable);
What?

Receive a variable and returns whether or not it is not defined.

Returns?

True if the variable is not defined, false otherwise.

Example?
var x; var y = 2; isDefined(x); // true isDefined(y); // false isDefined(someNotDeclaredVariable); // true
Responsive Bind
responsiveBind(handler, [initialize = false]);
What?

A function to handle responsive window size change, Triggered on document.ready, window.load and window.resize events.

Returns?

-

Parameter Name Expected Type Default Value Description
handler Function The function to handle responsive changes
initialize Boolean false Wheter or not to execute the handler once immediately
Example?
responsiveBind(function() { // run some dynamic fixes to the DOM to fix the current screen size $("some-element").css("font-size", $(window).width() / 100 + "px"); });

Important note: Use this solution only if there is no acceptable CSS solution as Javascript handlers consume much more system resources than CSS.

Extract Parameter
extractParam(test, defaultValue, [minIndex, [maxIndex]]);
What?

Extracts a parameter from the calling function.

Returns?

The parameter value or the default value if parameter was not found.

Parameter Name Expected Type Default Value Description
test String / Function A string to specify a parameter type to be extracted.
A function to receive the element and determine if it is the searched parameter
defaultValue Mixed The value to return in case a matching parameter was not found
minIndex Number 0 A minimal index to start the search from in the list of calling function parameters
maxIndex Number Infinity A maximal index to end the search at in the list of calling function parameters
Example?
function test(a, b, c, d) { var number = extractParam("number", 1); var string = extractParam("string", "default"); var bigNumber = extractParam(function(value) { return value > 3; }, 4); alert(number + ', ' + string + ', ' + bigNumber); } test("nice", 5, 4); // will alert '5, nice, 4' test("nice"); // will alert '1, nice, 4'
