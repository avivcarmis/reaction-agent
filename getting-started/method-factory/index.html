Method Factory

The Reaction Agent comes with build in static factory to generate various common and simple functions, as specified below.

Not Empty
MethodFactory.notEmpty();
What?

Returns a function that receive a value and determines whether or not it is not an empty string.

Result?
function(val) { return val != ''; }
Log
MethodFactory.log(message);
What?

Returns a function that logs the message to the browser console.

Example?
$(document).click(MethodFactory.log('click')); // is equal to $(document).click(function() { console.log('click'); });
Debug
MethodFactory.debug(message);
What?

Returns a function that debug the message to the browser console.

Example?
$(document).click(MethodFactory.debug('click')); // is equal to $(document).click(function() { console.debug('click'); });
Error
MethodFactory.error(message);
What?

Returns a function that error the message to the browser console.

Example?

Alert
MethodFactory.alert(message);
What?

Returns a function that alerts the message.

Example?
